## Practice Files: How to Use

Standard naming: practice_<topic>.py

- practice_json.py: JSON skills (parsing strings/files, transforming, pretty dumping, validation)

### Setup
```bash
python -m pip install -r requirements.txt
```

### Run the tests
```bash
pytest -q
```

### Workflow
1. Open `practice_json.py` and implement the functions marked NotImplementedError.
2. Run tests to get feedback.
3. Use the docstrings for examples and learning goals.

### Notes
- Tests live in `tests/` and verify correctness.
- Keep function names/signatures unchanged.
